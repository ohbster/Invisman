Guide:

For "equality" use [table]?(key)=(value)

For "less than" use [table]?lt=(key),(value)

For "less than or equal to" use [table]?le=(key),(value)

For "greater than" use [table]?gt=(key),(value)

For "greater than or equal to" use [table]?ge=(key),(value)

To sort by column use sort=(key)

To sort in ascending/descending order use direction=(asc/desc)

Example: For ID greater than 20 and less than or equal to 30, sort descending by ID
